# 프로젝트 소개

**모꼬지 (모임의 옛말 )  “여러 사람이 모여 놀이판을 여는것”**


  꼬지는 같은 동네에서 비슷한 나이대와 취미를 가진 사람들을 쉽게 만날 수 있게 해주고, 사용자는 관심사와 위치 기반으로 모임을 탐색하고, 새로운 관계를 만들 수 있게 돕습니다. 간편한 모임 생성과 관리 기능을 통해 누구나 손쉽게 모임을 주최하고 참여할 수 있습니다.
